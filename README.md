## 📌 Project Overview
This project predicts passenger survival on the Titanic using machine learning techniques.  
The model was built and evaluated using the Kaggle Titanic competition dataset.

---

## 🔧 Steps Performed

- Data Cleaning
  - Handled missing values (Age, Embarked, Fare)
- Feature Engineering
  - Created FamilySize feature
  - Created IsAlone feature
- Encoding
  - Converted categorical features (Sex, Embarked)
- Model Building
  - Random Forest Classifier
- Model Evaluation
  - Local validation accuracy ~81%
  - Kaggle Public Leaderboard Score: **0.78468**
